# 📋 Customer Feedback Form

A simple and responsive **Customer Feedback Form** built using **HTML, CSS, and JavaScript**.
This project collects user feedback including ratings, preferences, and comments in a clean UI.

---

## 🚀 Features

* 🧾 User-friendly feedback form
* 📱 Responsive design (works on mobile & desktop)
* ⭐ Service rating system (radio buttons)
* ✅ Multiple selection options (checkboxes)
* 📊 Dropdown for feedback source
* ✍️ Additional comments section
* 🎨 Clean and modern UI using CSS

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (basic)

## 💡 How It Works

1. User enters:

   * Name
   * Email
2. Selects how they found the service
3. Rates the service
4. Chooses liked features
5. Writes additional feedback
6. Clicks **Submit**
